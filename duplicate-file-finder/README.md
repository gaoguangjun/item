# 重复文件检查工具

一个基于 Web 的重复文件查找和清理工具，支持大目录扫描、MD5 精确匹配和批量删除确认。

## 功能特点

- **目录空间统计** - 显示总文件数、文件夹数、总大小和文件类型分布
- **重复文件检测** - 使用 MD5 哈希精确识别内容相同的文件
- **可视化界面** - 简洁的 Web 界面，实时显示扫描进度
- **智能删除** - 支持批量选择删除，默认保留原始文件
- **报告导出** - 可导出扫描结果为文本报告

## 快速开始

### Windows 用户

双击 `run.bat` 启动服务器，然后在浏览器中访问 http://127.0.0.1:8000

### 手动启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务器
python main.py
```

## 使用说明

1. 在扫描目录框中输入要检查的文件夹路径
2. 点击"开始扫描"按钮
3. 等待扫描完成，查看统计信息和重复文件列表
4. 勾选要删除的重复文件（可使用"智能选择"快速选择标记的副本）
5. 点击"删除选中"确认删除

## 注意事项

- 删除操作不可撤销，请仔细确认
- 建议先备份重要数据
- 大目录扫描可能需要较长时间

## 系统要求

- Python 3.8+
- 现代浏览器（Chrome、Edge、Firefox 等）
